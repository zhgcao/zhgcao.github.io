Git的设置

Git支持Git、SSH、HTTP/HTTPS三种协议访问远程仓库，3钟协议的远程仓库地址格式不同，以 GitHub 为例：
Git协议：git://github.com/...
SSH协议：git@github.com:...
HTTP/HTTPS协议：http://github.com/... 或 https://github.com/...
